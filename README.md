## 🛠 安装与依赖
### 系统要求

| 系统         | 开发状态 |
| ---------- | ---- |
| Windows 11 | ✅    |
| archlinux  | ✅    |
| 其他         | ❓    |

### 安装步骤
1. 克隆仓库：
```bash
   git clone https://github.com/not-linyi/MyDeskPetCore
   cd MyDeskPetCore
   ```
2. 安装依赖（建议使用虚拟环境）：
```bash
   python -m venv 
   venv source venv/bin/activate # Windows: venv\Scripts\activate 
   pip install -r requirements.txt
   ```
3. 构建/运行：
```bash
   python DeskPet.py
   ```
